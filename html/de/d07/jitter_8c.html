static spx_int16_t compute_opt_delay ( JitterBuffer *  jitter)
static

Based on available data, this computes the optimal delay for the jitter buffer.

The optimised function is in timestamp units and is: cost = delay + late_factor*[number of frames that would be late if we used that delay]

Parameters
tbArray of buffers
late_factorEquivalent cost of a late frame (in timestamp units)
